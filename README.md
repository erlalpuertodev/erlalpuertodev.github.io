# **🏠 Home**
## **Welcome to my AWS Cloud Resume Challenge!**
This project is inspired by **Forrest Brazeal's Cloud Resume Challenge.** It's designed to showcase my skills in cloud technologies, DevOps practices, and web development by building and deploying a personal resume website using AWS services.

## **🧱 Project Overview**
**Goals:**
* Create a resume website hosted on AWS.
* Use modern cloud tools and practices (CI/CD, Infrastructure as Code, etc.).
* Demonstrate knowledge of AWS, Git, GitHub, and web development.

**Features:**
* Static resume website using HTML, CSS, and JavaScript
* Hosted using GitHub Pages
* Tracks website views using AWS DynamoDB
* Backend Lambda function built with Python
* Visitor counter displayed on frontend
* CI/CD pipeline with GitHub Actions

## **🔧 Tech Stack**

Category | Tools/Services Used
-- | --
Frontend | HTML, CSS, JavaScript
Backend | AWS Lambda (Python)
Database | AWS DynamoDB
API | AWS API Gateway
Hosting | GitHub Pages
CI/CD | GitHub Actions
Infrastructure | AWS IAM, AWS CLI
Version Control | Git & GitHub
