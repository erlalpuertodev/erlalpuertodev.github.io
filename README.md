# **🌩️ I Completed the Cloud Resume Challenge!**
Hi! I'm **Erl Herold C. Alpuerto**, and I'm proud to say that I have successfully completed the [Cloud Resume Challenge](https://cloudresumechallenge.dev/), a hands-on, full-stack cloud project originally created by **Forrest Brazeal**. This challenge pushed me to combine my cloud knowledge, programming skills, and DevOps practices to build something real - my personal resume site powered by AWS.

**🔗 Live Website: https://erlalpuertodev.github.io/**


**Task** | **What I Did**
-- | --
📄 Resume in HTML | I created a responsive, mobile-friendly resume using plain HTML and CSS.
🎨 Styled with CSS | Custom styling was added for a clean, professional look.
🌐 Hosted on the Web | The site is deployed using GitHub Pages.
☁️ Cloud Database (DynamoDB) | Used AWS DynamoDB to store the number of site visitors.
🐍 Lambda Function (Python) | Wrote a Python function that reads and updates the visitor count.
🔌 API Gateway | Integrated Lambda with API Gateway to expose a public HTTP endpoint.
🔒 CORS Configuration | Configured CORS so my frontend can successfully call the backend.
🧠 JavaScript Frontend Integration | Used JS fetch() to call the API and display the visitor count in real-time.
🔄 CI/CD Pipeline | Set up GitHub Actions to automate deployment on every commit.
🛡️ IAM Roles & Permissions | Created secure IAM roles for Lambda to access DynamoDB.
